# KK Tan | AI & Financial Automation Portfolio

Welcome to the repository for my professional portfolio. I am a **Finance Automation Specialist** focused on transforming manual finance processes into high-speed automated pipelines using Python, AI, and modern data engineering.

## 🚀 Overview

This project is a high-performance, glassmorphism-styled portfolio website that showcases my expertise in financial automation, data processing, and AI implementation. It serves as a central hub for my projects, learning resources, and utility tools.

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3 (Tailwind CSS inspired custom variables), JavaScript (Vanilla)
- **Design**: Glassmorphism, Tech-Grid Backgrounds, Ambient Glow Effects
- **Automation Tools**: Python (Pandas, Polars, OpenPyXL), VBA, n8n, Ollama (Local LLMs)
- **Deployment**: Optimized for GitHub Pages

## 📂 Project Structure

- `index.html`: The main landing page featuring my profile, stats, project modules, learning resources, and contact information.
- `learning.html`: A dedicated page for curated tutorials and guides on Python, AI, and automation.
- `tools.html`: A utility page hosting custom-built tools like the [Markdown Editor](https://kktan1991.github.io/markdown/).
- `enterprise-automation.html`: Detailed breakdown of enterprise-level finance automation projects.
- `data-master-pro.html`: Showcase of the Streamlit-based data processing engine.
- `excel-binary-grouper.html` & `excel-data-grouper.html`: Specialized Excel automation utilities.
- `multi-threaded-processor.html`: High-performance data processing implementation.

## ✨ Key Features

- **Interactive UI**: Typewriter effects, smooth scrolling, and responsive mobile navigation.
- **Project Modules**: A bento-grid display of my key automation achievements and their business impact.
- **My Learning Hub**: Curated resources including "AI & Automation Masterclass" YouTube series and guides on MCP, where the place I upskilling my AI automation skills.
